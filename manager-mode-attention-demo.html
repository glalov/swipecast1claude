Interaction Design

Drawing attention to “Manager Mode”

Three premium, on-load treatments for the nav button. Each plays a short, polite animation when the page loads, then settles so it stays discoverable without looping forever.

1 Spotlight Sweep Recommended · most premium A soft amber pill with a glossy light that sweeps across twice — the Stripe/Apple “new feature” feel. Subtle, settles into a faint highlight.
2 Breathing Glow Pill A teal→amber gradient pill that gently breathes (scale + glow) three times, then rests as a solid teal highlight. High visibility, still tasteful.
3 Notification Ping + Sparkle An expanding ring “ping” radiates out, the label springs in, and an amber sparkle twinkles — playful, signals “something new.”
